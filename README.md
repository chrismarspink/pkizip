# PKIZIP

전자서명 · 암호화 · 압축 PWA

> **Live:** https://chrismarspink.github.io/pkizip/

## 기능

- **파일 압축** — tar.gz 아카이브
- **전자서명** — 파일에 서명 추가
- **공개키 암호화** — 수신자 지정 암호화 + 서명
- **비밀번호 암호화** — 비밀번호만으로 파일 잠금
- **인증서 관리** — 자체서명 인증서 발급, 로고 이미지 포함
- **다중 키 관리** — 여러 키를 이름으로 관리
- **오프라인 동작** — 네트워크 불필요

## 시작하기

```bash
npm install --legacy-peer-deps
npm run dev
```
